## Component Breakdown

### 1. Header Section
- Title with icon
- Info alert with usage tips
- Full-width across the page

### 2. Main Editor Column (Left - 8/12 width on desktop)
- **Previous Context Accordion**: Collapsible, shows last 3 story parts
- **Prompt Input**: Large multiline text field with placeholder examples
- **Character Selector & Action Buttons**: Character dropdown + Generate/Retry/Clear
- **Quick Templates**: Chip buttons for common prompt patterns
- **Generation Progress**: Shows when AI is generating, with detected context notes
- **Generated Content**: Rich text editor (Georgia serif font)
- **Action Buttons**: Save Draft / Insert into Story
- **Feedback Panel**: Revision instructions with templates

### 3. Sidebar Column (Right - 4/12 width on desktop)
- **Side Notes Panel**: 
  - Scene type dropdown
  - Tag chips (clickable to toggle)
  - Free-form notes textarea
- **Branching Panel**: 
  - Description of feature
  - Create Branch button (opens dialog)
- **History Panel**: 
  - List of previous generations
  - Each with timestamp, prompt preview, content preview
  - Restore button for each entry

## Responsive Behavior

- **Desktop (lg+)**: Two-column layout as shown above
- **Tablet (md)**: Still two columns, narrower spacing
- **Mobile (sm-)**: Single column, sidebar moves below main content

## Color Scheme

- Uses Material-UI theme (supports dark/light mode)
- Primary color for action buttons
- Secondary color for clear/cancel actions
- Info alerts: blue
- Success alerts: green
- Error alerts: red

## Interactions

1. **Generate Flow**:
   - User enters prompt
   - Clicks Generate
   - Progress bar appears
   - Status updates in real-time
   - Content appears in editor
   - Can edit, retry, or save

2. **Revision Flow**:
   - User sees generated content
   - Enters feedback
   - Clicks Generate Revision
   - New version replaces old
   - Old version saved to history
   - Can restore previous versions

3. **Branch Flow**:
   - User clicks Create Branch
   - Dialog opens
   - Enters branch name and prompt
   - Branch generates and saves
   - Main draft remains unchanged

4. **Draft Flow**:
   - User works on content
   - Clicks Save Draft
   - Draft saved with all metadata
   - Can continue editing later
   - Or insert into story when ready
